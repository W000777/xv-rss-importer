Import posts from the xvideos.com RSS feed.

== Description ==

A Wordpress plugin to import posts from the xvideos.com RSS feed.

== Installation ==

1. Upload the `xvideos-rss-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on RSS
